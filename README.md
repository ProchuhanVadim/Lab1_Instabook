# Instabook Social Network

Веб-додаток соціальної мережі, розроблений на Flask. Система дозволяє користувачам реєструватися, авторизуватися, відновлювати пароль, створювати публікації, залишати коментарі та редагувати особистий профіль.

## Технології

* **Мова програмування:** Python 3.x
* **Веб-фреймворк:** Flask
* **Аутентифікація:** Flask-Login
* **Безпека:** Werkzeug (хешування паролів)
* **База даних:** SQLite
* **Frontend:** HTML5, CSS3, Bootstrap 5

## Реалізований функціонал

### Для користувачів

* Реєстрація нового облікового запису.
* Авторизація та вихід із системи.
* Редагування особистого профілю.
* Відновлення пароля через одноразове посилання.

### Для контенту

* Створення нових публікацій.
* Перегляд стрічки публікацій.
* Коментування записів інших користувачів.
* Відображення автора та дати створення публікації.

### Безпека

* Зберігання паролів у вигляді хешів.
* Захист сторінок профілю, створення постів та коментування для неавторизованих користувачів.
* Одноразові токени для відновлення пароля.

## Кроки запуску

### 1. Клонування репозиторію

```bash
git clone <посилання_на_репозиторій>
cd BLOG_APP
```

### 2. Встановлення залежностей

```bash
pip install -r requirements.txt
```

### 3. Запуск застосунку

```bash
python app.py
```

### 4. Відкриття в браузері

```text
http://127.0.0.1:5000
```

## База даних

* users
* posts
* comments
* password_resets
